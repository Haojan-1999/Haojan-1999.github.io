电车难题思考

人自身是目的，而不是工具。——康德

前几天在B站看了罗翔老师和某个国外大学对电车问题讨论的视频，其实在此之前我在法律方法课上也了解过相关知识，只是在这之后我对这个问题又有了新的想法。

我们先来看看电车问题的三个版本（描述不一定完全准确）。

Version 1
假设你是电车驾驶员，你的刹车坏了，更不幸的是，在你的轨道上有五个人正在工作，你所驾驶的电车将会撞死他们，但是，你发现还是一个分支轨道，上面只有一个人在工作，除了选择是否更换轨道外没有其他选择。问：你会怎么做？

Version 2
你站在天桥上，看到有一台刹车损坏的电车。在轨道前方，有五个正在工作的人，他们不知道电车向他们冲来。一个体重很重的路人，正站在你身边，你发现他的巨大体形与重量，正好可以挡住电车，让电车出轨，不致于撞上那五个工人。你是否应该动手，把这个很胖的路人从天桥上推落，以拯救另外五个工人，还是应该坐视电车撞上那五个工人？

Version 3
有一天张三在医院，突然一个医生叫住他，说：”我这边有五个病人，他们都是各大国的元首，身患重病，一旦有一个人去世，都会引发世界动乱，一个人需要肾、一个人需要肝、一个人需要肺、一个人需要心脏、一个人需要脾，我看你身体不错，骨骼精奇，能否舍身捐献一下？”这时候，作为一个半身不遂、智力有缺陷、孤苦伶仃，生活无望的张三，你是否认为他应该捐献自己的器官？

以上三个问题，传统的思考方式是归结为道德道义论和目的论的讨论。

以康德为代表的道义论主张判断一个行为的正当性在于考虑其行为动机是否符合道德要求，而以边沁为代表的目的论则主张应从行为的结果判断行为的正当性。

如果按照道义论，则电车难题的答案就会是应选择牺牲五个人，而如果是目的论，则答案就会变成选择牺牲一个人。这里，有关道义论和目的论的理论论述，还可以阅读相关著作进一步学习了解，此处仅是个人粗浅理解。

对于电车难题，个人还有以下思考：

  • 关于生命价值能否比较的问题
    具体到电车难题，问题的核心在于生命价值的比较，即五个人生命是否比一个人的生命更重要？高地位的人是否比一般人的生命价值更高？
    如果按照当今人权理论，人人生而平等，生命价值无所谓贵贱，那么我们就回到了道义论价值体系中。

  • 个人立场是否影响价值判断
    一般来说，对于电车问题，版本一多数人选择宁愿选撞死一个人，而在版本二、三，人们却又不会选择去牺牲一个人。
    可见，人们在道义论和目的论的立场上可以是动态变化的。个人认为造成这样的原因是人出于不同立场来保障个人利益。在版本一中，人们被默认为是即将犯罪既遂，因此人们出于对自己利益的考量，自然选择可能会判处较轻处罚的结果。而在版本二、三，人们是站在旁观者的立场，没有作为的义务，即不对五人死亡负责。而这时选择让一个人去代替五个人，自然让自己也牵涉到最后的责任之中。这里，我们也不排除完全从道德的角度做出不作为选择的人。

当今网络时代，一些人会站在”道德”制高点指责他人，无论是当事人出于道德还是目的，这些人都会从与之相反的目的和道德进行指责，毕竟指责一下也不用负责。尽管我们这个社会赞美主动牺牲、甘愿奉献的品质，但是人总是自私的，更好地生存下去是生物的本能选择。

少一些苛责，多一些包容；少一些疯狂，多一些理性。因为道德是一扇明镜，而不是一套枷锁。

反对解释所运用的逻辑学规则

反对解释，即从法律条文规定的内容来推论出反面的结果。

其逻辑学规则表示为：若p，则q，其反对解释为若非p则非q。从其逻辑学规则来看，反对解释并不能保证其结论总是成立。

因此，其使用条件为：

（1）法条所确定的条件为法律效果的全部要件；
（2）法律规定所确定的条件为法律效果的必要要件。[1]

个人对此的一个粗浅理解，即从逻辑学的角度看，想要保证反对解释成立，其原命题的逆命题必须成立。

以具体法条为例：

《刑法》第49条规定“犯罪的时候不满十八周岁的人和审判的时候怀孕的妇女, 不适用死刑。”

其反对解释为“犯罪的时候满十八周岁的人或审判的时候不怀孕的妇女, 适用死刑“，显然是荒谬的，因为其逆命题“不适用死刑的人是犯罪的时候不满十八周岁的人和审判的时候怀孕的妇女”是不成立的。

相反，

“过失犯罪的, 法律有规定的才负刑事责任”

其反对解释“过失犯罪的，法律没有规定的不负刑事责任”成立，因为其逆命题“过失犯罪的，负刑事责任是因为法律有规定”也是成立的。

[1]:张明楷.罪刑法定原则与法律解释方法[J].华东刑事司法评论 (第三卷) [A].法律出版社, 2003.24.
